# 📝 자바스크립트 게시판 미니 프로젝트 (프론트엔드)

이 프로젝트는 **React**를 활용하여 백엔드(Spring)에서 구현된 게시판의 프론트엔드를 담당합니다.  
사용자는 **게시글 목록 조회, 게시글 작성, 수정, 삭제, 댓글 기능**을 웹 UI를 통해 사용할 수 있습니다.  

---

## 📌 프로젝트 환경
- **프레임워크**: React (JSX 기반)
- **상태관리**: React Hooks (useState, useEffect)
- **라우팅**: React Router
- **스타일링**: Tailwind CSS
- **API 통신**: Axios (RESTful API)
- **아이콘**: Lucide-react
